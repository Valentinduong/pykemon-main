# Pykemon

Projet réalisé dans le cadre du Projet Technique Spécialisé – Informatique (ISEP).

## Description
Application développée en Python suivant une conception orientée objet.
Le projet simule un jeu de type Pokémon avec des combats, un inventaire et des mécaniques de jeu.

## Fonctionnalités
- Gestion des combats
- Gestion de la santé
- Inventaire
- Menu interactif en console

## Technologies utilisées
- Python
- Programmation orientée objet

## Lancement du projet
Exécuter le fichier principal :
```bash
python Main.py
